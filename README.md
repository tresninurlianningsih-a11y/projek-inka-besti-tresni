# Projek Inka Besti Tresni 🌟

Proyek ini adalah halaman web interaktif sederhana dengan desain Glassmorphism modern yang dibuat khusus untuk Besti Tresni.

## 🚀 Fitur Utama
* **Glassmorphism Design**: Tampilan antarmuka transparan dengan efek blur latar belakang dan bayangan dinamis.
* **Glow & Hover Effects**: Mikro-animasi interaktif saat elemen disentuh oleh kursor.
* **Sapaan Interaktif**: Tombol cepat untuk menyapa Besti secara langsung.
